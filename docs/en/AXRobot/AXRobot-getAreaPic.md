# Get Area Basemap

## `getAreaPic(areaId) -> {Promise.<any>}`

Get area map image

### Parameters

| Name | Data Type | Description |
| -------- | -------- | -------- |
| `areaId` | string | area ID |

### Return value `Promise.<any>`

Regional map image data stream

### Example

```javascript
...
const mapData = await axRobot.getAreaPic("<areaId>");
...
````
