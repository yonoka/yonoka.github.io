# Simple Motion

## `motionFor(type) -> {void}`

Robot motion control

### Parameters

| Name | Data Type | Description |
| ------ | --------------------------------- | -------- |
| `type` | [MotionType](#/Define-MotionType) | motion type |

### Return Value

none

### Example

```javascript
...
axRobot.motionFor(MotionType.Forward);
...
````
