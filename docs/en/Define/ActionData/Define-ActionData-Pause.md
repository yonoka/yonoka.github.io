# ActionData - Pause

Pause Action Parameters

## Attributes

| Name | Data Type | Description |
| ------------- | -------- | ---------------------------------- |
| `pauseTime` | number | Pause time, unit: seconds<br/>0 means no pause |

## Example

```javascript
{
   "type": ActionType.Pause,
   "data": {
     "pauseTime": 20
   }
}
````
