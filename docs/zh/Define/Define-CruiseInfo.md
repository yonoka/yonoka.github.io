# CruiseInfo

巡游信息

## 属性

| 名称         | 数据类型    | 说明     |
| ------------ | ----------- | -------- |
| `id`         | number      | 巡游标识 |
| `name`       | string      | 巡游名称 |
| `businessId` | string      | 业务标识 |
| `sites`      | []CruisePoi | 巡游点位 |
| `remark`     | string      | 备注     |



<br/>
<br/>
<br/>
CruisePoi - 巡游点位

## 属性

| 名称      | 数据类型 | 说明     |
| --------- | -------- | -------- |
| `poiId`   | string   | poi 标识 |
| `poiName` | string   | poi 名称 |
