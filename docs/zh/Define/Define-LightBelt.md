# LightBelt

灯还控制

## 属性

| 名称     | 数据类型                            | 说明                         |
| -------- | ----------------------------------- | ---------------------------- |
| `mode`   | number                              | 执行端模式，1-上位机，2-底盘 |
| `color`  | [LightColor](#/Define-LightColor)   | 灯带颜色，见LightColor       |
| `indexs` | [LightIndex](#/Define-LightIndex)[] | 分段灯带数组                 |
