# ServerInfo

服务器信息

## 属性

| 名称      | 数据类型 | 说明                                                              |
| --------- | -------- | ----------------------------------------------------------------- |
| `ip`      | string   | 地址                                                              |
| `type`    | number   | 服务类型 <br/> 0：私有化<br/>1：国际版<br/>-1：关闭私有化或国际版 |
| `offline` | number   | 0: 非纯离线<br/> 1: 纯离线                                        |
